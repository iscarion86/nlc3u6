# Repó klónozás
git clone https://github.com/iscarion86/nlc3u6.git

pip install -r requirements.txt

# "Build" folyamat
A Pythonnál egy compile futtatása szükséges, ami az alábbi paranccsal történik:
python -m compileall .

A szöveges utasítások halmaza ezzel fordított gépi kóddá alakul.

# Webszerver alkalmazás indítása
python app.py
